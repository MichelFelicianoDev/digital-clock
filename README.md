# Relógio Digital

> 📌 Este projeto foi desenvolvido seguindo o tutorial da [Larissa Kich](https://github.com/Larissakich) no YouTube ([@larissakich](https://www.youtube.com/@larissakich)), como prática de HTML, CSS e JavaScript. Estou retomando a programação após um tempo afastado da área, então a maior parte da estrutura segue o tutorial — usei o processo para revisar lógica e entender o funcionamento na prática.

Um relógio digital simples que exibe horas, minutos e segundos em tempo real, feito com HTML, CSS e JavaScript puro.

## Funcionalidades

- Exibe a hora atual no formato HH:MM:SS
- Atualização automática a cada segundo
- Formatação com zero à esquerda (ex: 09 em vez de 9)

## Como usar

1. Clone o repositório:
```bash
   git clone https://github.com/MichelFelicianoDev/digital-clock.git
```
2. Abra o arquivo `index.html` no navegador

Não é necessário instalar nada — o projeto não usa Node.js nem gerenciador de pacotes.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
